# Superpowers
---

Superpowers is a complete software development methodology for your coding agents, built on top of a set of composable skills and some initial instructions that make sure your agent uses them.

## Why Recommend It?
Superpowers offers a suite of out-of-the-box skills that fit into the development ecosystem, giving users an easy point of introduction into skill usage. One of the skills /subagent-driven-development also offers a lightweight framework to users who want to split larger, complex tasks into independant subagents to preserve individual agent context windows.
